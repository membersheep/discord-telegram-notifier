# Discord<->Telegram

## Installation
```npm install```

# Telegram Configuration
- Speak to ```@BotFather``` on telegram
- Create a new bot with ```/newbot```
- ```/setjoingroups``` ENABLED
- ```/setprivacy``` DISABLED
- Put the bot token into ```config.json```
- Put the chat id into ```config.json```
